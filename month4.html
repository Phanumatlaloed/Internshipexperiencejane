เรื่องราวการฝึกงานเดือนที่ 4

เดือนสุดท้ายของการฝึกงาน — ลงมือสร้าง Portfolio Website จริงด้วย Elementor บน Obtech Theme พร้อมส่งมอบผลงานและแบ่งปันความรู้ให้ทีม

ภาพรวมงานที่ได้รับมอบหมาย

  • ทดลองใช้ Elementor บน Obtech Theme เพื่อออกแบบ Portfolio Website
  • วางโครงสร้างหน้าเว็บและคิด UI/UX ด้วยตนเองตั้งแต่ต้น
  • ปรับแนวคิดจาก "รับงานออกแบบ" → "Portfolio ส่วนตัว"
  • ตั้งค่า WordPress แบบ Open Source + ฐานข้อมูลผ่าน phpMyAdmin
  • ส่งงานให้พี่เลี้ยงตรวจและนำ Feedback ไปปรับปรุง
  • สอนงาน/ให้คำปรึกษาเพื่อนร่วมงานในช่วงรอรับงานใหม่

กระบวนการพัฒนาเว็บไซต์

เริ่มจากการวางแผนโครงสร้างเว็บไซต์ก่อน จากนั้นเลือก Elementor ที่คุ้นเคยจากประสบการณ์ Month ก่อนหน้า ทำให้ทำงานได้ไวขึ้น มีความเข้าใจในการใช้สี การจัดวาง Layout และฟังก์ชันของธีมอย่างครบถ้วน

เรียนรู้หลัก: ออกแบบเว็บ Portfolio จริง + แบ่งปันความรู้ให้ทีม

สิ่งที่ได้เรียนรู้

  • การออกแบบ Portfolio Website ด้วย Elementor + Obtech Theme
  • การวางโครงสร้างเว็บและ UI/UX ด้วยตนเองตั้งแต่ต้น
  • การตั้งค่าระบบฐานข้อมูล phpMyAdmin + MySQL
  • การแก้ไขปัญหาเฉพาะหน้าและรับ Feedback ไปปรับปรุง
  • การสอนงานและถ่ายทอดความรู้ให้เพื่อนร่วมทีม

ปัญหาที่พบ

  • เซิร์ฟเวอร์มีความไม่เสถียรบางช่วง ต้อง Backup ข้อมูลอยู่เสมอ
  • มีข้อเสนอแนะเรื่อง Design Detail บางส่วนจากพี่เลี้ยง
  • ต้องอาศัยการฝึกฝนต่อเนื่องเพื่อเพิ่มความคล่องตัว
  • การปรับแนวคิด Rebrand กลางคันต้องวางแผนใหม่บางส่วน
